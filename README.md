# Automation suite created for running tests in Windows, Mac and Linux 
# To run the tests 
		Windows - you have to right click on the wintests.testng.xml and run as TestNG
				- Or update the maven pom file with the wintests.testng.xml file and run as mvn test
		Mac - you have to right click on the mactests.testng.xml and run as TestNG
			- Or update the maven pom file with the mactests.testng.xml file and run as mvn test
		Linux - you have to right click on the linuxtests.testng.xml and run as TestNG
			- Or update the maven pom file with the linuxtests.testng.xml file and run as mvn test
